Getting started

Installation:

 
npm install @devo/alerts-api-client

Using the client:

Create class that implements IConfig interface. This class will retrieve the authentication token:

import { IConfig } from "src/clients/alertsClient";

export class AuthConfig implements IConfig {
    getAuthorization() {
      return lt.user._metadata.credential.standAloneToken;
    }
  }

Create an instance of the client and example of how to retrieve the list of alerts:

import { Client } from '@devo/alerts-api-client';

const apiUrl = 'https://api-us.devo.com/alerts/';

const authConfig: AuthConfig = new AuthConfig();

const alertsClient = new Client(authConfig, apiUrl);
const alerts = alertsClient.getListByCriterias('100', '0', '1651053600000', '1651057200000', 'id', true, true);

console.log(alerts);
